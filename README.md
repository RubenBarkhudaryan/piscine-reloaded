# piscine-reloaded

## Overview
This repository contains my 42 piscine-reloaded exercises, grouped from ex00 to ex27.

The set revisits core C and shell concepts through short, focused tasks.

## Contents
- Basic shell utilities
- Intro C functions and output helpers
- Pointer and parameter manipulation
- String and memory-related basics
- Recursion and algorithmic mini-tasks
- Headers, macros, and small Makefile tasks
- File display exercise and utility routines

## Build and Run
Most exercises are standalone files or tiny units.

For exercises with Makefiles:
- make
- make clean
- make fclean

For standalone C files:
- cc -Wall -Wextra -Werror file.c -o program
- ./program

## Goal
Practice and reinforce fundamentals through constrained, norm-compliant implementations.

## Notes
This repository is educational and follows 42 piscine constraints and style.
